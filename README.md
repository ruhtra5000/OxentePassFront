## 📍 Venda e Divulgação de Eventos em Pernambuco

Este projeto, desenvolvido para a disciplina de **Programação Web**, visa ser uma plataforma focada no **turismo regional em Pernambuco**, facilitando a **divulgação de eventos** e a **venda de ingressos** para o público local e visitantes.

> **Nota:** Este repositório contém a aplicação **Frontend** do projeto. 

<br>

## 💻 Tecnologias Utilizadas

⚙️ **Frontend**

* **Framework/Biblioteca:** React (Next.js)
* **Linguagem:** TypeScript
* **Estilização:** Tailwind CSS
* **Ícones:** Lucide React

<br>

## 🚀 Como Executar o Projeto

Para rodar a plataforma localmente na sua máquina, siga os passos abaixo:

### 1. Clonar e Instalar
Abra o terminal, clone o repositório e instale as dependências:

```bash
# Clone este repositório
git clone [https://github.com/ruhtra5000/OxentePassFront.git](https://github.com/ruhtra5000/OxentePassFront.git)

# Acesse a pasta do projeto
cd OxentePassFront

# Instale as dependências
npm install

# Gere o build de produção
npm run build

# Inicie o servidor
npm run start

```
